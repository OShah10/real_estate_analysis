## Running re_retirement_rank.py
This Python script is used for preprocessing the data and compute senior suitability score for the properties in our datasets.
It outputs the transformed dataset to Suitability_score_house.csv (this file is already populated in the repository)

### To Run:
python re_retirement_rank.py

## Running prop_score_testing.py
This Python script is used for evaluating models and running CLI recommendations

### To Run:
python prop_score_testing.py

## Running recommender_knn.py
This Python script is used for evaluating normalized KNN and running CLI recommendations

### To Run:
python recommender_knn.py

## Running Demo App
streamlit run Path\To\Repository\real_estate_analysis\demo_app.py

## Architecture & File Map

```
project/
│
├── re_retirement_rank.py          # STEP 1 — Score every property, save CSV
├── recommender_knn.py             # STEP 2 — KNN model
├── prop_score_testing.py          # STEP 3 — Evaluate models, run recommendations
├── demo_app.py                    # STEP 4 — Streamlit UI
│
├── Suitability_score_house.csv    # Generated by re_retirement_rank.py
│
└── Datasets/
    ├── realtor-data.zip.csv                              # Core housing listings
    └── (Copy) VT/
        ├── usa_zi_premium_crimerisk.csv                  # Crime risk per zip
        ├── usa_zi_2024 crimerisk-statistics.csv          # Crime stats (min/max)
        ├── usa_zi_2024_demographic_estimates.csv         # Median rent, demographics
        ├── usa_zi_base_currentyear.csv                   # Healthcare scores per zip
        ├── usa_zi_base_currentyear-statistics.csv        # Healthcare stats (min/max)
        ├── usa_zi_premium_environment.csv                # Temp, air quality, walkability
        └── usa_zi_premium_environment-statistics.csv     # Environment stats (min/max)
```

## File Reference

| File | Role | Reads | Writes |
|---|---|---|---|
| `re_retirement_rank.py` | Pipeline step 1 — score all properties | Raw CSVs from `Datasets/` | `Suitability_score_house.csv` |
| `recommender_knn.py` | Pipeline step 2 — KNN evaluation | `Suitability_score_house.csv` | None |
| `prop_score_testing.py` | Pipeline step 3 — evaluate and recommend (CLI) | `Suitability_score_house.csv` | None |
| `demo_app.py` | Streamlit UI | `Suitability_score_house.csv` | None |
